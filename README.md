# Fentris

A MonoGame-based block stacking game inspired by the TGM series. Features all-original modes and backgrounds (made out of my artistic photograms) and a custom rotation system. 

**This is a work in progress, due to university and other projects i haven't had much time to work on this, especially since the new "graded" mode requires a lot of balancing that i have to ask better players for advice on. It is playable though!**
